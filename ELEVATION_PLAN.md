# Project Elevation Plan: Building A Handwritten Digits Classifier

**Theme:** Computer Vision

## 1. Vision & Goal
Upgrade from basic ML to a 'Digit Recognition API' using CNNs, data augmentation, and rigorous evaluation.

This plan aligns with the portfolio-wide goal of elevating projects from basic tutorials to Senior/Principal-level data science products characterized by rigorous experiment design, research-grade reporting, and software engineering best practices.

## 2. Elevation Backlog

The following phases outline the concrete steps required to elevate this project. They should be executed sequentially.

[ ] **Phase 1: Architecture Upgrade** - Replace basic models with a Convolutional Neural Network (CNN) architecture.
[ ] **Phase 2: Data Augmentation** - Implement image augmentation strategies to improve robustness.
[ ] **Phase 3: Comprehensive Metrics** - Add per-class F1 scores, confusion matrices, and inference latency measurements.
[ ] **Phase 4: Ablation Study & Report** - Write a paper-style report documenting the impact of architecture and augmentation choices.
[ ] **Phase 5: Inference API** - Deploy the model as an API suitable for form-digit extraction workflows.
